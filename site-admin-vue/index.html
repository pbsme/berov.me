Панель управления

Цель — создать интерактивную систему с помощью компонентов на Vue 3. Компоненты взаимодействуют между собой через props и $emit. Всего сайт состоит из 11 компонентов vue + index.vue. На сайте есть экран логина, меню с индикацией активного экрана и несохраненных данных, всплывающие уведомления, экран редактирования объявления, менеджеров сайта и контактов.
